regn.data module¶

class regn.data.GprofData(path, batch_size=None, surface_type=-1)¶

Bases: torch.utils.data.dataset.Dataset

Pytorch dataset for the Gprof training data.

This class is a wrapper around the netCDF4 files that are used to store the GProf training data. It provides as input vector the brightness temperatures and as output vector the surface precipitation.
