Datasets and -interfaces¶

The regn.data module contains interfaces to the datasets that we are working with in the project.

GPROF data¶

Pytorch datasets¶

GprofData(path[, batch_size, surface_type])

Pytorch dataset for the Gprof training data.

Reading binary data¶

read_file(f)

Read GPM binary file.

check_sample(data)

Check that brightness temperatures are within a valid range.

write_to_file(file, data[, samples])

Write data to NetCDF file.

create_output_file(path)

Creates netCDF4 output file to store training data in.

extract_data(base_path, y, m, d, samples, file)

Extract training data from binary files for given year, month and day.
