REGN¶

Robust Estimation of Global Precipitation using Neural Networks (REGN) is a research project investigating how quantile regression neural networks (QRNNs) can be used to yield more reliable and potentially more accurate precipitation estimates from the satellite observations of the Global Precipitation Measurement (GPM).
